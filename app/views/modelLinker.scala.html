NetLogo Web Model Link Creator

There are two kinds of links you can create to share or embed your NetLogo model using NetLogo Web. Shareable links open the full NetLogo Web interface and are best for sending to someone who wants to run and explore the model. They're great for email, social media, or course pages. Embeddable links load a standalone, borderless view of the model suited for embedding in an iframe on your own website. Embeddable links also support version pinning, which lets you lock a link to a specific NetLogo Web release for reproducibility.

See the Hosting and Sharing documentation for more details on each link type.
