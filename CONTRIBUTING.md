# Contributing to Galapagos / NetLogo Web

For full contributing guidelines — including git workflow, testing, publishing Tortoise changes, and all code style guides (CoffeeScript, CSS, Scala) — see the **[NetLogo Web Contributing Guide](https://github.com/NetLogo/Tortoise/wiki/CONTRIBUTING)** in the Tortoise wiki.

## Galapagos-Specific Resources

The [Galapagos wiki](https://github.com/NetLogo/Galapagos/wiki) has documentation specific to this project:

- **[Home](https://github.com/NetLogo/Galapagos/wiki/Home)** — project overview and team
- **[Building and Running](https://github.com/NetLogo/Galapagos/wiki/Building-and-running)** — local setup instructions
- **[Technical Details](https://github.com/NetLogo/Galapagos/wiki/Technical-Details)** — architecture and internals
- **[NetLogo Web Appearance and Styling](https://github.com/NetLogo/Galapagos/wiki/NetLogo-Web-Appearance-and-Styling)** — UI and CSS conventions
- **[NetLogo Web Release Process](https://github.com/NetLogo/Galapagos/wiki/NetLogo-Web-Release-Process)** — how releases are cut
- **[Updating the Models Library](https://github.com/NetLogo/Galapagos/wiki/Updating-the-Models-Library)** — how to update bundled models

Feel free to submit a pull request to improve any of these documents.
